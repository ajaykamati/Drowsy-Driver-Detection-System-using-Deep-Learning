🚗 Drowsy Driver Detection System using Deep Learning

This project presents a Drowsy Driver Detection System developed using deep learning and computer vision techniques to improve road safety by detecting driver fatigue in real time. It uses OpenCV and Mediapipe to track facial landmarks, especially eye movements, and identifies drowsiness based on eye closure and blinking patterns. When signs of fatigue are detected, the system triggers an alert sound to warn the driver and help prevent accidents. Implemented in Python, the system provides real-time monitoring, accurate detection, and an efficient alert mechanism, making it a practical solution for reducing accidents caused by driver drowsiness.
